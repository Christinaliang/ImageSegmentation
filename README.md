# Image Segmentation

## Requirements:

+ python 2.7

+ numpy

+ OpenCV

## Method

Image segmentation uses K-Means

## Useage

```bash
python main.py --cluster <number of clusters> --thresh <threshold for error rate> --input <image path>
```

If you want to use default arguments, you can use:

```bash
python main.py
```

Default:

+ cluster: 5

+ thresh: 0.0001

+ input: image.jpg
